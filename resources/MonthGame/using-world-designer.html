Features that are not complete will be described in italics.

Controls

Place a wall by clicking on the screen. Change the orientation of the wall using your mouse wheel. Delete walls by right clicking on them.

Load

You can load a map as a starting point by selecting File | Load and navigating to the file. The default map in the game is called "wavy-world.map" and it is embedded in the jar file. If you replace this file with something of your own and place that in the jar instead you can change the default game map.

Save

Save your work by selecting File | Save. You can then load your map into the game by selecting Game | Load Map in the game window.

Clear

Start with a clean slate by selecting File | Clear surface. Any unsaved work will be lost.

Playing your Map

To play a game with your map first save it and then load it from the game window using Game | Load map.
